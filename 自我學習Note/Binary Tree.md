## Binary Tree
![](https://imgur.com/Rf5aZHp.jpg)

二元樹 
 - 可以為空的樹 
 - 不為空則具有 Root，左、右子樹 
 - 各節點與分支度介於 0 與 2 之間 
 - 左、右子樹具有次序之分(order tree) 

二元樹的種類 
 - 偏斜二元樹 Skewed Binary Tree:可分左、右偏斜樹 
 - 完滿二元樹 Full Binary Tree 
 - 完整二元樹 Complete Binary Tree 
## Binary Search Tree
可應用於資料的排序跟搜尋。

如何建立BST:輸入的資料皆與樹根比，比樹根跟小的放至左子樹，比樹根大的放至右子樹。
